# 2048 Game

## Overview
This is a Python-based implementation of the classic 2048 game. The game is played on a 4Ã—4 grid, where players merge tiles with the same numbers to reach the 2048 tile. The objective is to strategically combine tiles by moving them in different directions.

## Features
- Classic 2048 gameplay
- Simple and intuitive controls
- Score tracking
- Random tile generation after each move
- Game over detection

## Technologies Used
- **Programming Language**: Python
- **Libraries**: NumPy, Tkinter (for GUI, if applicable)

## Installation

### Prerequisites
Ensure you have Python installed on your system.

### Clone the Repository
```bash
git clone https://github.com/HackSmith010/2048_game.git
cd 2048_game
```

### Run the Game
```bash
python game.py
```

## How to Play
1. Use arrow keys (Up, Down, Left, Right) to move tiles.
2. When two tiles with the same number collide, they merge into one.
3. The goal is to create a tile with the number 2048.
4. The game ends when no moves are possible.
